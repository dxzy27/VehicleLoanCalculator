# Vehicle Loan Calculator

## Overview

The **Vehicle Loan Calculator** is an Android application designed to help users calculate the monthly payment, total loan amount, total interest, and total payment for a vehicle loan based on inputs like vehicle price, down payment, loan period, and interest rate. This app also provides details about the author and a link to the GitHub repository for the project.

## Features

- **Vehicle Price Input**: Users can enter the price of the vehicle.
- **Down Payment Input**: Users can enter the down payment amount.
- **Loan Period Input**: Users can specify the loan period in years.
- **Interest Rate Input**: Users can input the interest rate.
- **Loan Calculation**: The app calculates the loan amount, monthly payment, total interest, and total payment based on the inputs.
- **About Page**: Provides information about the app, the author, and a link to the GitHub repository.

## Technologies Used

- **Programming Language**: Java
- **Android SDK**: Android Studio for app development
- **UI Framework**: Android Views (TextViews, EditTexts, Buttons, ImageViews, etc.)
- **App Architecture**: Simple UI-driven app with logic for loan calculations

## How to Use

1. Clone or download the repository.
2. Open the project in **Android Studio**.
3. Build and run the project on an emulator or physical device.
4. Enter the required inputs (vehicle price, down payment, loan period, and interest rate).
5. Click **Calculate** to view the loan details.
6. Navigate to the **About** section to learn more about the app and the author.

## Installation

To get started with this project, follow these steps:

### Prerequisites

- **Android Studio** (latest version)
- **Java Development Kit (JDK)** installed

### Steps to Install

1. Clone the repository to your local machine:
    ```bash
    git clone https://github.com/your-username/Vehicle-Loan-Calculator.git
    ```

2. Open Android Studio and select **Open an existing Android Studio project**.

3. Navigate to the cloned repository and open the project.

4. Build and run the app on an emulator or your physical device.
