# Vehicle Loan Calculator

## Overview

The **Vehicle Loan Calculator** is an Android application developed to calculate the loan amount, total interest, total payment and monthly payment for a vehicle loan when the users put in inputs like vehicle price, down payment, loan period, and interest rate. This app also provides an about page to display details about the author and a link to the GitHub repository for the project.

## Features

- **Vehicle Price Input**: Enter the price of the vehicle.
- **Down Payment Input**: Enter the down payment amount.
- **Loan Period Input**: Enter the loan period in years.
- **Interest Rate Input**: Enter the interest rate in %.
- **Loan Calculation**: Calculates the loan amount, total interest, total payment and monthly payment based on the inputs.
- **About Page**: Provides author informations, and a link to the GitHub repository.

## How to Use

1. Open the project in **Android Studio**.
2. Build and run the project on an emulator or physical device.
3. Enter the required inputs (vehicle price, down payment, loan period, and interest rate).
4. Click **Calculate**.
5. Outputs (loan amount, total interest, total payment and monthly payment) are displayed. 
6. Navigate to the **About** section to learn more about author and the link to the GitHub repository.
